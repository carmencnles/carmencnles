## 📈 Proyecto Destacado

### 🌲 [Análisis y Clasificación de Incendios Forestales mediante Matriz de Confusión](https://github.com/carmencnles)
Proyecto desarrollado durante el Curso de Especialización en IA y Big Data enfocado en la evaluación de modelos predictivos para la detección de incendios forestales.
* **Objetivo:** Analizar el rendimiento de un modelo de clasificación, optimizando las métricas de evaluación para reducir los falsos negativos (incendios no detectados), un factor crítico en escenarios de riesgo real.
* **Qué incluye:** Procesamiento de datos climáticos/geográficos, cálculo y visualización de la **Matriz de Confusión**, y análisis de métricas clave (*Precision, Recall, F1-Score y Accuracy*).
* **Tecnologías utilizadas:** `Python` *(Pandas, NumPy, Scikit-Learn, Matplotlib/Seaborn)*.
